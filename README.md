# BookList

## List of Features
- HttpURLConnection.
- AsyncTask.
- JSON Parsing.
- ArrayAdapter.
- ListView.

# Description
- Search books by author or title. 
- Retrieves data from Google Books API. 
- Only the first 10 results will be displayed.
